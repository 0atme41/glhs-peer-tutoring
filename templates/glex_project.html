What is Project Glex?

Food deserts are regions that have little access to a nearby sources of food. This is a problem in low-income areas, where access to the time or transportation necessary to make a long trip to a grocery store may be severely limited. Project Glex is a solution for organizations to map possible food deserts within a specified zip code. Each grocery store is represented on the map as a circle with a radius of half a mile, allowing organizations like charities and municipalities to pinpoint where to direct aid. We hope this project can be used as another tool in the arsenal of charities to provide aid to our communities and build up poor areas.

How did we make Project Glex?

Project Glex uses Folium, a flexible Python mapping library, to plot, and a Google API to locate stores. We also use several other smaller libraries to make everything work smoothly and intuitively.

Who made Project Glex?

Project Glex was created by Sam Fishburn and Paul Boucher. If you have any questions or suggestions, feel free to email Sam at sjfishburn@students.wcpss.net. We're thinking of expanding the project's scope to include data like heat islands and average income, so we would love to hear your feedback!
